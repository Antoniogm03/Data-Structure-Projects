Introducción

En esta práctica realizaremo los TDA necesarios para crear el juego de las cifras y las letras, para ello utilizaremos diferentes contenedores de la STL, como lo serán map, set, vector, etc.

Ejecutables

conjunto_palabras:

Para probar el funcionamiento de nuestro TDA LettersSet.

conjunto_palabras <FicheroEntrada> <palabra>

Parameters
<FicheroEntrada>Ruta al archivo con información sobre el conjunto de letras.
<palabra>Palabra de la que calcular la puntuación.

bolsa_letras:

Para probar el funcionamiento de nuestro TDA LettersBag.

conjunto_palabras <FicheroEntrada>

Parameters
<FicheroEntrada>Ruta al archivo con información sobre el conjunto de letras.

Para ello:

  • Creamos un conjunto de letras (LettersSet), y lo rellenamos a partir de la información leída del archivo que pasamos como parámetro.
  • Creamos una bolsa de letras (LettersBag), y la rellenamos con la información del LettersSet anterior (usamos constructor por parámetros).
  • Extraemos todas las letras de la LettersBag (aleatoriamente) y las imprimimos por pantalla.

palabras_longitud:

Para probar el funcionamiento de nuestro TDA Dictionary.

conjunto_palabras <FicheroEntrada> <Número>

Parameters
<FicheroEntrada>Ruta al archivo a un fichero con las palabras de un diccionario.
<Número>Entero con la longitud de las palabras que buscamos.
  • Construimos un Dictionary con el fichero de las palabras, extrae de dicho diccionario las palabras de la longitud que buscamos y las imprime por pantalla.

cantidad_letras:

Para probar el funcionamiento de nuestro TDA Dictionary.

conjunto_palabras <FicheroDictionary> <FicheroLetras>

Parameters
<FicheroEntrada>Ruta al archivo a un fichero con las palabras de un diccionario.
<FicheroLetras>Ruta al archivo a un fichero con las letras.
  • Construimos un Dictionary con el fichero de palabras, un LettersSet con el fichero de letras, e imprime por pantalla, para cada letra en el LettersSet, el número de ocurrencias de la letra en el diccionario y su frecuencia relativa.
